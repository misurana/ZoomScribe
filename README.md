# ZoomScribe

[![React 19](https://img.shields.io/badge/Frontend-React_19-blue?style=for-the-badge&logo=react)](https://react.dev/)
[![Zoom SDK](https://img.shields.io/badge/Integration-Zoom_SDK-orange?style=for-the-badge&logo=zoom)](https://developers.zoom.us/)

ZoomScribe is a professional meeting assistant that integrates with the Zoom Meeting SDK. It provides real-time transcription, automated scribing, and report generation in React.

## Features
- Real-time Transcription: Capture words as they are spoken.
- AI-Powered Scribing: Automated note-taking and summarization.
- PDF Report Generation: Download comprehensive summaries.
- React 19: Built with the latest React features.

## Tech Stack
- Frontend: React 19, TypeScript
- SDK: Zoom Meeting SDK
